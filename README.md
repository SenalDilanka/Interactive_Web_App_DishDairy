# Interactive_Web_App_DishDairy

The application, known as "Dish Diary," is an interactive web application designed to assist users in exploring, managing, and organizing different types of dishes and recipes through a simple and user-friendly interface. The application enables users to view a collection of dishes, access information regarding individual recipes, and maintain a personalized record of their favorite dishes.

The application is developed using modern web technologies, including HTML, CSS, Bootstrap, and JavaScript, which provide a responsive design and a smooth user interface for users. The application interface is developed with a focus on usability, enabling users to easily navigate through different parts of the application interface.

The application includes a number of important functionalities, which enable users to view information regarding individual dishes, organize content, and provide interactive functionality through JavaScript. The application focuses on design, content organization, and interactive functionality, which provide users with a smoother experience. The application, "Dish Diary," can be considered a practical application for managing different types of food-related content, as well as a demonstration of web development concepts, including responsive design, DOM, and JavaScript.

## Features

- View dishes
- Add new recipes
- Interactive UI using JavaScript
- Login

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run

1. Download the project
2. Open index.html in browser
